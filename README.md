# AI Resume Screening System

An AI-powered Resume Screening System that automates resume analysis and candidate ranking using Natural Language Processing (NLP) and Machine Learning techniques.

The system extracts text from uploaded PDF resumes, compares them with job descriptions using TF-IDF Vectorization and Cosine Similarity, and ranks candidates based on relevance scores.

Built using Python, Streamlit, Scikit-learn, and NLP techniques.
